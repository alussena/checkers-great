# Checkers Pro – шашечная платформа нового поколения

Полнофункциональное веб-приложение для игры в русские шашки, созданное как прототип стартапа.  
Ориентировано на удержание пользователей и монетизацию.

## 🔥 Ключевые возможности

- ♟️ **Русские шашки** – взятие назад, дамки ходят на любое расстояние.
- 🤖 **ИИ-соперник** – 3 уровня сложности (easy/medium/hard) на базе минимакса с альфа-бета отсечением.
- 🌐 **Мультиплеер в реальном времени** – игра с другом по ссылке или быстрый поиск соперника (WebSockets).
- 🧠 **AI-тренер** – после партии ИИ анализирует каждый ход и объясняет ошибки.
- ⏱️ **Быстрые дуэли** – режим "3 минуты на партию" для тренировки стратегического мышления.
- 🎨 **Темы оформления** – классическая, чёрно-белая, зелёная, розовая. Сменные скины фигур (Pro).
- 💬 **Чат** – свободный в приватных играх, быстрые фразы в публичных.
- 👑 **Лидерборд по городам** – рейтинг ELO, топы игроков Алматы, Астаны и др.
- 🔐 **Авторизация** – Google OAuth через Supabase.
- 💳 **Upgrade to Pro** – Stripe-подписка открывает премиум-темы, ИИ-тренера и максимальную сложность.

## 🚀 Технологический стек

- **Frontend**: Next.js, React, Tailwind CSS, Socket.io-client
- **Backend**: Node.js, Express, Socket.io, Supabase (PostgreSQL, Auth)
- **AI**: Minimax с альфа-бета отсечением
- **Платежи**: Stripe Checkout

## 🧠 Бизнес-модель

- Freemium: базовый функционал бесплатен.
- Pro-подписка ($4.99/мес): премиум-темы, AI-тренер, hard-уровень ИИ, расширенная статистика.
- Потенциал: платные турниры, продажа скинов, корпоративные лиги.

## 📦 Запуск локально

1. Клонируйте репозиторий
2. В папках `client` и `server` выполните `npm install`
3. Создайте `.env` по образцам `.env.example`
4. Запустите сервер: `cd server && npm run dev`
5. Запустите клиент: `cd client && npm run dev`
6. Откройте http://localhost:3000
